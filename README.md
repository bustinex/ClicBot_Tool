# ClicBot Dev Tools

Desktop application for debugging, programming, and driving ClicBot robots.  
Includes assembly mode, module structure inspection, animation editor, joystick control, ZIP upload, and advanced developer utilities.

## What is it?

ClicBot Dev Tools is a desktop application for debugging, programming, and driving ClicBot robots:

- Assembly mode and module structure inspection
- Animation Studio
- Joystick drive and programming wizard
- ZIP upload and play/stop
- WebSocket bridge
- SSH tool
